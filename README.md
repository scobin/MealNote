MealNote
=================================
食事の内容を記録するアプリ。

機能：<br>
○　時間、料理名、場所、値段、評価、人物、写真、メモという内容を記録する。<br>
○　記録内容の閲覧及び編集ができる。<br>
○　集計機能。評価ランキング、食べた回数ランキング。<br>
○　気に入り。写真の閲覧。
